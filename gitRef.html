Git Referance Sheet

Git Command Line and Directories

$ cd Navigates to the desired drive.

$ cd "directory location" Navigates to desired directory.

$ git ls-filesLists all files in the repository, including staged files.


Initializing a local Git Repository

$ git init Create a new local repository.


Getting recent code from GitHub

$ git pull Pulls the latest version of code from another repository.


Initializing local Git repository from GitHub

$ git clone https://github.com/YOUR-USERNAME/YOUR_REPOSITORY Clones the GitHub repository to a local machine. **Make sure the command is in the drive you wish repository to be located in.


Adding and Commit Changes to Respository

$ git add . Get everything ready to commit.

$ git commit -m "Message" Commit changes.


Checking Repository Status

$ git status Checking the current state of the repository.


Saving Changes to GitHub Repository

$ git add * Adds files to staging.

$ git commit -m "Enter Message"Commits all changes to directory.

$ git push Pushes most recent local copy to GitHub


Viewing Commits to a Repository

$ git log Lists all the commits made in the repository in reverse chronological order.


Making a Temporary Branch from Master Repository

$ git branch -av Lists all exsisting braches.

$ git branch Creates a new branch.


Merging Branch into Master

$ git checkout master Switches command to master branch.

$ git merge Merges changes from another branch.
